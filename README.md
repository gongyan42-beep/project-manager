# 🚀 项目管理器

## 这是什么？
一个可视化的项目管理工具，让您轻松管理所有开发项目。

## 项目类型
🌐 **网页应用** - 有界面，在浏览器操作

## 功能
- ✅ 自动扫描 `/Users/gusuping/code/` 下的所有项目
- ✅ 查看项目列表和详细信息
- ✅ 一键启动/停止项目
- ✅ 查看项目运行状态（运行中/已停止）
- ✅ 快速访问项目网址
- ✅ 查看端口分配情况
- ✅ 显示项目技术栈

## 快速开始（2步）

### 第1步：启动项目管理器
```bash
cd /Users/gusuping/code/project-manager
./start.sh
```

### 第2步：打开浏览器
自动打开浏览器访问：http://localhost:5003

## 使用说明

### 查看项目列表
- 打开页面后自动显示所有项目
- 显示项目名称、端口、状态、描述

### 启动项目
1. 找到要启动的项目
2. 点击 "启动" 按钮
3. 等待启动完成（会显示加载动画）
4. 点击 "访问" 按钮打开项目

### 停止项目
1. 找到正在运行的项目
2. 点击 "停止" 按钮
3. 确认停止

### 查看项目详情
- 点击项目名称查看 README
- 查看项目配置信息
- 查看部署状态

## 端口信息
- 本地开发：http://localhost:5003
- 服务器部署：http://manager.longgonghuohuo.com

## 技术栈
- 后端：Flask (Python)
- 前端：HTML + Tailwind CSS + JavaScript
- 进程管理：subprocess

## 常见问题

### Q: 为什么有些项目显示"无法启动"？
A: 可能是项目没有 `start.sh` 文件，或文件没有执行权限

### Q: 如何停止服务？
A: 运行 `./stop.sh` 或在页面上点击停止按钮

### Q: 项目状态不准确怎么办？
A: 刷新页面，系统会重新检测所有项目状态
